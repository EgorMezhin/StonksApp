# StonksApp

**App testing conditions:**
-----------------------------------
- iPhone 11
- iOS 14.2
- portrait orientation

**Description:**
-----------------------------------
The App allows you to see the cost of the most "active" shares on the stock exchange. 

**App features:**
-----------------------------------
- MVC
- Pod "Charts"
- API iexcloud.io
- GET reques
- UIAlertController
- UIPickerView
